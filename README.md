# Robotic Dog

## Project Overview
This project focuses on developing an **Robotic Dog** designed to assist operators in surveillance, inspection, and safety tasks. The robot can navigate complex environments, detect obstacles, and safely interact with humans, all while providing real-time telemetry and mission feedback.

The system combines **autonomous navigation, 3D mapping, mission planning, and safety features** to enable efficient monitoring with minimal human intervention. Users can also manually override the robot when needed.

## Key Features
- **Autonomous Navigation:** Dynamic path planning and obstacle avoidance.  
- **3D Mapping:** Real-time visualization of the environment.  
- **Safety Management:** Emergency behavior handling, cliff detection, and human proximity awareness.  
- **Mission Control:** Multi-step mission orchestration and remote control interface.  
- **Flexible Charging & Status Indicators:** USB-C and magnetic docking, with LED indicators for operational status.  

## Hardware Overview
The robot integrates a **Raspberry Pi 5**, **STM32 microcontroller**, **sensors (IMU, ultrasonic, infrared)**, **servo motors**, and power management systems to support mobility, navigation, and safety features.

## Purpose
The goal of this project is to **enhance security operations** by automating routine surveillance tasks while ensuring safety, flexibility, and real-time data collection. It provides a practical platform for testing autonomous robotics concepts in real-world scenarios.
